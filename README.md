# Autopkgtest Runner

Frontend for the autopkgtest Debian test framework.

Built with GTK4/Adwaita. Part of the [Danne L10n Suite](https://github.com/yeager/debian-repo).

## Installation

### Debian/Ubuntu
```bash
sudo apt install autopkgtest-runner
```
